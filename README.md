# Sitio Web - Napoli FC 🔵

Proyecto de dos páginas HTML sobre el club de fútbol **S.S.C. Napoli**, creado como parte del Trabajo Práctico N°2 del Laboratorio de Programación.

---

## Estructura del proyecto

```
tp2/
├── index.html       → Página principal
├── pagina2.html     → Datos curiosos
└── estadionapoli.jfif  → Imagen del estadio (local)
```

---

## Páginas

### `index.html` — Página principal
- Bienvenida al sitio con título en color azul marino sobre fondo gris claro (`whitesmoke`).
- Descripción general del club: historia, colores, estadio, palmarés y cantidad de hinchas.
- Menú de navegación con:
  - Enlace interno a `pagina2.html`.
  - Enlace externo a ESPN para estadísticas actualizadas del equipo.
- Imagen del Estadio Diego Armando Maradona (ruta local).
- Imagen embebida en base64 de la ciudad de Nápoles.
- Cierre con mensaje de agradecimiento en verde.

### `pagina2.html` — Datos curiosos sobre el Napoli
- Fondo celeste (`#f0f8ff`) con título en rojo oscuro.
- Tres secciones temáticas:
  - **La Era Maradona y su Legado:** retiro del dorsal 10, su rol como ícono de Nápoles y la conexión argentina en los campeonatos.
  - **Historia y Fundación:** origen anglo-italiano en 1926 y el significado del color azul cielo.
  - **Récords y Logros:** máximo goleador (Dries Mertens), récord de victorias, posición histórica en el sur de Italia y recuperación tras la quiebra de 2004.
- Enlace de regreso a la página principal.

---

## Tecnologías utilizadas

- HTML5
- CSS en línea (estilos aplicados directamente con el atributo `style`)

---

## Cómo visualizar el sitio

1. Descargar o clonar los archivos del proyecto.
2. Asegurarse de que `index.html`, `pagina2.html` y la imagen `estadionapoli.jfif` estén en la misma carpeta.
3. Abrir `index.html` con cualquier navegador web moderno.

> **Nota:** la imagen del estadio usa una ruta local (`C:\Users\Usuario\Desktop\tp2\estadionapoli.jfif`). Para que se vea correctamente en otra computadora, se debe actualizar esa ruta o reemplazarla por una imagen en la misma carpeta del proyecto.
